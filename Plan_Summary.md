# Vocational Transition Survey — 작업 가이드

## 1. 프로젝트 목표

한국 직업교육 정책 타임라인 설문(`Screenshot/SurveyPannel.png`, `SurveyPannel2.png`)의 각 정책 이벤트에 대해:

- (a) **정책 방향 전환 정도** 1–5 점수 (1: 연장 ↔ 5: 방향전환)
- (b) **정책 영향력** 1–5 점수 (1: 매우제한 ↔ 5: 지대함)
- (c) **의미에 대한 본인 견해** 서술

을 작성해야 한다. 이를 돕기 위해 모든 이벤트의 원자료를 표준 마크다운으로 정리(Phase 1)한 뒤, 횡적 비교를 통해 점수·견해 초안을 산출(Phase 2)한다.

## 2. 워크플로우

| 단계 | 사용자 | Claude |
|---|---|---|
| Phase 1 | 우측 패널 텍스트를 `.txt`로 `Screenshot/`에 저장(여러 개 일괄 가능) | `Events/{YYYY}_{단축명}.md`로 변환, 인덱스 표 갱신 |
| Phase 2 | "분석해줘" 요청 | 모든 Events 횡적 비교 → 각 MD 하단 `## 분석 제안` 채우기, frontmatter 점수 갱신 |
| 응답 | 제안 검토·조정 후 설문 입력, MD `status: reviewed`로 갱신 | — |

**원칙**: Phase 1이 충분히 모인 뒤 Phase 2 진행(상대 비교가 가능해야 점수가 의미를 갖는다).

## 3. 폴더 구조 & 파일명 규칙

```
Vocational_Transition_Survey/
├── Plan_Summary.md          ← 본 파일
├── Screenshot/              ← 원자료 (사용자 투입)
│   ├── SurveyPannel.png
│   ├── SurveyPannel2.png
│   └── *.txt                ← 이벤트별 원본 텍스트
└── Events/                  ← Phase 1 산출물
    └── {YYYY}_{단축명}.md
```

**파일명 규칙**: `{YYYY}_{단축명}.md`
- 연도 prefix로 OS 정렬 = 시간순.
- 단축명은 띄어쓰기 없이 정책 핵심 단어 1–3개 (예: `1963_산업교육진흥법.md`).

## 6. Phase 2 점수 척도 가이드

설문 패널(`Screenshot/SurveyPannel.png`)에 명시된 1–5 척도 기준. 상대 비교 결과를 기반으로 분포 보정.

**정책 방향 전환 정도** (1: 연장 ↔ 5: 방향전환)
- 5점 (방향전환): 영역 자체의 신설 또는 작동 원리 전면 교체 — 직업교육·훈련의 새 패러다임 출범 수준
- 4점: 새 트랙·새 작동 원리 출범 등 한 시대를 닫고 다음 시대를 여는 중대 단절
- 3점: 기존 체제 위에 새 영역·새 우산을 더하는 확장적 개편
- 2점: 위층 추가, 운영 단위 확장, 부분 재편 등 점진적 보완
- 1점 (연장): 시행령급 조정·기존 정책의 미세 정비

**정책 영향력** (1: 매우제한 ↔ 5: 지대함)
- 5점 (지대함): 수십 년에 걸친 후속 정책 경로 전체를 규정, 현재까지 단절 없는 작동
- 4점: 한 시대(10–20년) 정책 흐름을 형성한 모법, 후속 정책 다수의 직접 토대
- 3점: 5–10년 정책 흐름, 후속 정책의 부분 토대 또는 일부 분야 안정 정착
- 2점: 제한적·국지적 효과, 단독 영향력은 작지만 누적적 기여
- 1점 (매우제한): 진행형이거나 실효성 미확정, 단기·국지적 효과에 그침

## 7. 자료 전달 팁 (Phase 1)

1. **포맷**: 우측 패널 텍스트를 그대로 복사해 한 이벤트당 `.txt` 1개로 저장. 기존 `산업교육진흥법 ….txt`와 동일 포맷이면 변환이 자동화된다. 줄 순서·머리표(`○`) 그대로 두면 된다.
2. **파일명 자유**: 본문에서 연도·제목을 추출해 `{YYYY}_{단축명}.md`로 새로 명명한다.
3. **분량**: 한 번에 5–10개씩 넘기면 효율적. 너무 적으면 변환 호출이 잦고, 너무 많으면 한 번에 검토가 어렵다.
4. **누락 허용**: 본문에 깨졌거나 누락된 항목이 있어도 그대로 주면 빈 섹션으로 보존한다(추측으로 채우지 않음).
5. **카테고리 첫 줄**: 좌측 카테고리(중등직업교육 / 고등 및 평생직업교육 / 직업훈련 / 진로교육 / 자격제도·자격체계)가 텍스트 *첫 줄*에 오도록. `category` 매핑의 핵심.
